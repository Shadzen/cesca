Система мониторинга климатических условий серверной стойки и управления доступом AURA

Предназначена для отслеживания параметров доступа и мониторинга климатических условий в серверном шкафу.

Для реализации мониторинга к контроллеру подключаются датчики, сигналы которых обеспечивают оперативное реагирование на внештатные ситуации. Контроллер AURA‑MS‑CTR мгновенно отправляет данные с датчиков на сервер системы.

Обслуживающий персонал оперативно получает информацию об инцидентах по средствам оповещения через интерфейс ПО.

Система преобразует все полученные данные в электронные архивы в виде наглядных таблиц и графиков.
